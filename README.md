# 🏥 Hospital Management System

A full-stack Hospital Management System built using React, Node.js, and MySQL.  
This application allows users to manage hospital operations like booking appointments, viewing doctors, tracking billing, and managing patient records.

---

## 🌐 Live Demo

🔗 Frontend:  
https://hospitalmanagementshravani.netlify.app/

🔗 Backend API:  
https://hospital-management-system-1-jrj5.onrender.com

---

## 🚀 Features

- 🔐 User Authentication (Login with JWT)
- 👨‍⚕️ View Doctors List
- 📅 Book Appointments
- 📋 View & Cancel Appointments
- 💰 Billing Management
- 📊 Dashboard with real-time stats
- 🎯 Filter appointments (date & type)

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js
- MySQL

### Deployment
- Frontend: Netlify
- Backend: Render

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/shravani492006/Hospital-Management-System.git
cd Hospital-Management-System

**Backend setup**
cd hospital-backend
npm install
npm start

**Fronted Setup**
cd hospital-frontend
npm install
npm start

🔑 Environment Variables
Create a .env file inside hospital-frontend:
REACT_APP_API_URL=https://hospital-management-system-1-jrj5.onrender.com
